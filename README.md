# rasalhag-press
Official repository for my coloring book portfolio. High-quality vector art for people who still use physical crayons. Strictly anti-AI, hand-drawn, and prone to extreme artistic perfectionism.

## Project Structure
- **/assets**: High-quality vector illustrations and source graphic assets.
- **/data**: Catalog and metadata (CATALOG.json).
- **/docs**: Technical documentation, style guides, and project roadmap.
