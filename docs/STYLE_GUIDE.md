# Style Guide

Visual identity and design standards for rasalhag-press.

## Color Palette
- **Canvas (Background)**: `#FDFCFB` (Soft warm-white)
- **Structural Panels**: `#F8F6F3` (Light gray)
- **Border Rules**: `#E5E2DD` (Fine-line gray)
- **Accent/Dashed**: `#D1CCC5` (Architectural grid)

## Typography
- **Headings**: Inter (Sans-serif, clean)
- **Body/Display**: Cormorant Garamond (Serif, elegant italics)
- **Metadata**: JetBrains Mono (Monospaced, technical)

## Visual Elements
- **Lines**: Razor-thin border rules to mimic ink drafting.
- **Frames**: Dashed structural outlines for catalog mockups.
