# Contributing to rasalhag-press

Thank you for your interest in contributing! Please note that this is a curated portfolio project.

## Guidelines
- **Strictly Anti-AI**: All contributions must be 100% hand-drawn vector art. AI-generated assets will be rejected.
- **Perfectionism**: We value high-quality lines and manual vectorization.
- **Focus**: Contributions should align with the project's aesthetic roadmap.

## How to contribute
If you want to suggest improvements or changes:
1. Open an Issue to discuss your idea.
2. Fork the repository and create a new branch.
3. Submit a Pull Request.

All contributions are reviewed to maintain the project's vision.
